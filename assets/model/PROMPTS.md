# 模型能力图标 Prompts

> 规格：1024×1024 PNG，透明背景。实际 UI 中按 18-32px 使用。
> 统一遵循 `assets/PROMPT_PREFIX.md` 的故宫像素风、16 色调色板和透明 PNG 规范。

本组由 `$imagegen` 生成 4×2 contact sheet 后拆分落盘，用于 Model 面板的能力徽章与连通测试。

## 通用生成 Prompt

见 `assets/attachments/PROMPTS.md` 中同一批 contact sheet prompt。

## 输出文件

- `model-vision.png`：视觉能力 / 已通过图片输入测试。
- `model-text.png`：文本能力 / 文本连通测试。
- `model-test-ok.png`：连通测试成功。
- `model-test-fail.png`：连通测试失败。
