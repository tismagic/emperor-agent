# Clawd Tank Assets

These SVG animation assets are copied from `marciogranzotto/clawd-tank` at
commit `6f0093b40e3714db24615409bf31c351f66114f7`.

They are used by the optional Emperor Agent desktop pet companion. Only the
MIT-licensed visual assets are vendored here; the ESP32 firmware, SDL2
simulator, BLE daemon, and menu bar app are not included.

See `LICENSE` in this directory for the upstream MIT license.
